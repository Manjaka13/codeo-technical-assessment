# codeo-test-part1

Technical assessment for Codeo, part 1. 

#### Project Setup

```sh
npm install
```

#### Compile and Hot-Reload for Development

```sh
npm run dev
```

#### Compile and Minify for Production

```sh
npm run build
```

#### Run Unit Tests with Vitest

```sh
npm run test:unit
```

#### Lint with ESLint

```sh
npm run lint
```

#### Format with Prettier

```sh
npm run format
```
